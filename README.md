# 🌱 Vedant Ghatole Portfolio

> "Every journey begins with a seed."

A modern, interactive portfolio website built with a unique **Seed → Tree Growth Journey 🌱🌳** concept.  
This project represents my journey in **Computer Engineering, AI, and Software Development**.

---

## 🧠 Concept

This portfolio is designed as a **storytelling experience**, where each section represents a stage of growth:

- 🌱 Seed → Introduction  
- 🌿 Roots → Education  
- 🌱 Sprout → Involvement  
- 🌳 Growth → Skills  
- 🍃 Branches → Projects  
- 🍁 Leaves → Achievements  
- 🌳 Trunk → Experience  
- 🌌 Future → Vision  

---

## ✨ Features

- 🎨 Modern UI/UX (Dark Theme + Gradient Accents)
- ⚡ Smooth animations & micro-interactions
- 📱 Fully responsive design
- 🌐 Interactive project showcase
- 📊 Animated stats & achievements
- 🔗 Social & contact integration
- 🧭 Sticky navigation with smooth scroll

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, React, TypeScript  
- **Styling:** Tailwind CSS  
- **Animations:** Framer Motion  
- **3D/Visuals:** Three.js (optional)  
- **Deployment:** Vercel
