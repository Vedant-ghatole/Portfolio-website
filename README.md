# 🌱 Seed to Tree Portfolio – Vedant Ghatole

An interactive storytelling portfolio that represents my life journey as a **tree growing from a seed**.

The seed was planted on **April 13, 2005**, and as the user scrolls through the website, the seed grows into a sprout, plant, and finally a strong tree representing my **skills, projects, and achievements**.

This portfolio combines **cinematic storytelling, modern UI/UX, and interactive animations** to showcase my growth as a **Computer Engineering student and developer**.

---

# 🌳 Concept

The portfolio is designed as a **growth journey**:

```
Seed → Roots → Sprout → Plant → Branches → Leaves → Full Tree
```

Each stage represents a part of my life:

| Stage       | Meaning                          |
| ----------- | -------------------------------- |
| 🌱 Seed     | Birth – April 13, 2005           |
| 🌿 Roots    | Foundation & School Education    |
| 🌱 Sprout   | Early interests in technology    |
| 🌿 Plant    | Engineering journey              |
| 🌳 Branches | Skills & experience              |
| 🍃 Leaves   | GitHub projects                  |
| 🌞 Tree     | Present abilities & future goals |
